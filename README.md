# PlayZone

Initial implementation scaffold for the PlayZone multiplayer mini-games platform.

Current scope:
- shared room and player types
- in-memory room manager
- TicTacToe game engine
- React UI shell structure for the lobby/game flow

Next steps:
- wire Socket.IO server and client transport
- implement server-authoritative room state
- add Snakes & Ladders and Ludo engines
- add Snake canvas rendering
